Measuring the difference: a machine learning model to detect linguistic and topic bias

We already know that there exists a difference in terms of numbers between male and female biographies in Wikipedia. We also found an example where a non-neutral language and stereotypes are used to describe a famous female personality. Now, the question is: Are these issues a generality in Wikipedia or did we tried really hard to find some examples?

In order to quantify these differences we trained a machines learning model. This model takes as input different words appearing in the overview of the biographies and aims to predict the gender of the person being described. The idea behind it is that if both genders are described in the same way, the model wonâ€™t be able to predict better than random (it is 50% in a balanced dataset). On the other hand, if the model can do better, it means there is a pattern to learn from the data and that actually, the bias is there.

How did the model perform?

54% of the observations were correctly predicted, confirming that the bias does exist.

Is it a general result?

It is. After 50 iterations, the model's results variated only 0.001.

Is it true for every field of occupation?

No, it's not. For the military and reliction fields the model predicted a random score several times. Som we cannot state that there's a linguistic bias.
